# Customer Shopping Behaviour Analysis

## Project Summary
- This project explores customer shopping behaviour using transactional data to identify trends in revenue, subscriptions, product performance, and customer segments.
  The analysis is performed using SQL, and insights are presented through an interactive Power BI dashboard.

## What This Project Covers
- Revenue analysis across customer demographics
- Subscriber vs non-subscriber spending comparison
- Product and category performance analysis
- Customer segmentation based on purchase history

## Tech Stack
- SQL ( MySQL )
- Power BI
- Power Query

## Outcome
- The project provides actionable insights that help understand customer behaviour and support better business decisions related to marketing, pricing, and customer retention.
